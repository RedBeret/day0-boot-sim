# Security Policy

## Scope

`day0-boot-sim` is a local-only synthetic training repo. It should never contain:

- real credentials
- customer data
- proprietary images
- real device identifiers

## Reporting a Security Issue

If you find a security issue in the code or documentation, please report it privately before opening a public issue.

Include:

- what you found
- how to reproduce it
- the affected file or workflow
- any suggested mitigation

## Safe Disclosure Notes

- Do not post secrets in a public issue.
- Do not submit reports that include customer or proprietary data.
- If the problem is about unsafe sample data, replace it with synthetic values before sharing it.

## Supported Versions

This repo is currently maintained as a single moving mainline. The latest default branch content is the supported version.
