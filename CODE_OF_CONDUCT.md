# Code of Conduct

This project is meant to be practical, calm, and welcoming.

## Our Standards

Please help keep the space:

- respectful
- constructive
- curious
- patient with learners
- focused on the work, not the person

## Unacceptable Behavior

- harassment or personal attacks
- hate speech or discrimination
- repeated bad-faith arguing
- doxxing or threats
- posting real credentials, customer data, or proprietary artifacts

## Maintainer Responsibilities

Project maintainers may edit, remove, or reject content that makes the repo less safe or less useful. That includes behavior that disrupts collaboration or changes that break the repo's synthetic-only guardrails.

## Reporting

If something needs attention, open a private security-style report if the issue is sensitive. Otherwise, open a normal issue with enough context for maintainers to act on it.
